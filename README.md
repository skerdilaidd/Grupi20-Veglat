# Programi për Llogaritjen e Pagave

Ky është një projekt i vogël në C++ që llogarit pagat e punëtorëve bazuar në orët e punës. Projekti është realizuar për të demonstruar përdorimin e GitHub për menaxhimin e kodit burimor, bashkëpunimin përmes branches dhe pull requests.

## Funksionalitetet
- Futja e orëve të punës për 10 punëtorë
- Llogaritja e pagës për secilin punëtor
- Llogaritja e:
  - Pagës totale
  - Pagës mesatare
  - Pagës maksimale
  - Pagës minimale
- Validimi i të dhënave (0–48 orë)

## Teknologjitë
- C++
- Git & GitHub
